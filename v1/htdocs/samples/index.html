Kadecot API Tutorial

1. Just follow the links below and see what happens.







2. Download the following zip



3. Extract to some folder (call it %KSAMPLE_FOLDER% in the following)

4. Open index.html(this file) by your favorite text editor and modify KADECOT_IP to your Kadecot's IP address

5. Start web server with the documentroot set to %KSAMPLE_FOLDER%.
Web Server for Chrome is my recommendation.(Dead easy!)

6. Open index.html with your browser and check the samples works as before.

7. Since you have your files locally, you can modify the samples now. Enjoy!
