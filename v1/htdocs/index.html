Kadecot Control Panel

By Sony CSL

Kadecot|JS commands

  • Start Kadecot|JS foreground : kadecot startfg
  • Start Kadecot|JS background : kadecot start
  • Stop background Kadecot|JS : kadecot stop
  • Stop and start Kadecot|JS background : kadecot restart

Additional settings

  • A command line to build JSONP API URL Generator:
    cd ${HOME}/.kadecot/v1/htdocs/APIQueryGenerator/;npm install;npm run build
  • A command line to setup GotAPI server (usually runs on a smartphone) IP address Setup the server as open as possible!
    ${HOME}/.kadecot/v1/tools/initGotAPI.sh
  • External plugin setup tool

Apps

  • JSONP API URL Generator
    As mentioned above, please type 'cd ${HOME}/.kadecot/v1/htdocs/APIQueryGenerator/;npm install;npm run build' before you first access this app (may take long time..be patient!).
  • WAMP API Samples

Open Ports

  • WAMP : 41314
  • ECHONET Lite : 3610
  • WEBSOCKET GPIO : 41316
  • SOCKET GPIO : 41317
